# Junyi Academy — Claude Code Plugin Marketplace

均一教育平台的 Claude Code 插件市集。

A curated collection of Claude Code plugins by Junyi Academy.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add junyiacademy/claude-plugins
```

## Available Plugins

### English Coach

**Description:** Turn Claude Code into your English coach — get natural, idiomatic English coaching in every conversation while you code.

**Category:** Learning

**Install:**
```bash
/plugin install english-coach@junyiacademy-claude-plugins
```

**What you get:**
- Automatic English coaching in every message (English, Traditional Chinese, or mixed)
- Native speaker phrasing, concise alternatives, and grammar fixes
- Zero setup — just install and it works
- Sub-agents are not affected (no wasted tokens)

**Details:** [plugins/english-coach](plugins/english-coach)
